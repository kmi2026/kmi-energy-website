# 🌐 KMI 网站内容后台 (CMS) 操作说明书

这份说明书专为市场部、运营部及其他**非技术人员**编写。您无需看懂任何一行代码，只需像填写 Excel 表单一样，即可在全球范围内随时随地更新 KMI 官方网站上的中英文内容。

---

## 🔒 1. 管理员必读：如何给同事开通后台修改权限？
为了保证公司网站绝对的安全，**只有被管理员开通了权限的账号，才能在后台成功修改文字**。
如果您（目前唯一管理员）想把修改文字的工作交给团队其他成员，请照做以下步骤：

1. 让即将接手网站的同事花 1 分钟去注册一个免费的 [GitHub 账号](https://github.com/signup)。
2. 获取他们的 **GitHub 用户名 (Username)**。
3. 请您自己登录您的仓库设置页面：👉 [点击前往 Collaborators 页面](https://github.com/kmi2026/kmi-energy-website/settings/access)。
4. 点击右侧绿色的大按钮 **【Add people】**。
5. 在弹出的框里输入这名同事的 GitHub 用户名，并在下一步选中他们。
6. 让您的这名同事登录他的个人邮箱，点击里面那封 GitHub 蓝色邀请邮件里的 **"View invitation"**，并且在打开的网页点击绿色按钮接受邀请。

**恭喜！现在这名同事有了合法的“修改白名单钥匙”，下面他就可以跟着下面的第 2 步教程去愉快地修改网站文字了！**

## 2. 操作员教程：如何登录并修改大字？

### 步骤 A：打开后台专属网址
请在任意电脑的浏览器（推荐 Chrome 或 Edge）中输入以下地址：
👉 **后台入口：[https://kmi2026.github.io/kmi-energy-website/admin](https://kmi2026.github.io/kmi-energy-website/admin)**
*(注：如果加载偏慢请稍等，看到纯黑色的登录界面即代表访问成功)*

### 步骤 B：一键授权安全登录
1. 点击页面正中央唯一的按钮 **"Login with GitHub"**。
2. 此时弹出一个绿色的验证页面，询问是否允许向后台授权。请您点击底部的绿色大按钮 **"Authorize" (授权认证)**。
3. 授权通过，您非常顺利地进入了可视化的图形管理后台！

### 步骤 C：像填表一样修改文字
1. 在左侧深色菜单兰中，点击 **[Collections]** 下方唯一的 **[🌐 网站文案配置]** 选项卡。
2. 此时屏幕中间会出现两个集合：
   - 🇨🇳 中文文案库
   - 🇬🇧 英文文案库
3. 点击您今天要**调整的那门语言**。
4. 页面右侧会展开一个非常长的高级表单。为了方便您查找，所有文本框上方我都标记了**“中文拼音大标签”**（例如：`首页 - hero_h1` 、`业务1 (隔音) - biz1_title` 等）。
5. 找到需要修改的地方，用鼠标点击文本框，**把旧文字删掉，直接敲入您想展现的新文字即可！**（如果是大篇幅的介绍内容，您可以按回车键随便换行，多写几段话！）。

### 💾 步骤 D：点击保存，发布全局！
1. 当您修改完毕并核对无误后，把目光移向界面的**最右上角**。
2. 点击绿色的 **【Save】 (保存)** 大按钮。
3. 只要按钮立刻从 Saving 变成了 "Saved"，就代表您的修改内容已经光速提交给了服务器！
4. **大功告成，您现在就可以关掉后台浏览器页面了。**

## ⏳ 3. 常见问题：我改完了，网页什么时候能看到变化？
- 您在后台点击过【保存】后，并不是一秒生效。因为在这个时间间隙里，云端机器人生在自动花 **大概 1 到 2 分钟左右** 重新打包生成最新、最快的缓存网页静态包。
- 您稍微倒杯水，**2分钟后**回到网站正式主页：👉 [https://kmi2026.github.io/kmi-energy-website/](https://kmi2026.github.io/kmi-energy-website/)
- 在键盘上按下 **`F5`** 键（或者 Ctrl+R / Command+R）强制刷新一次网页缓存。
- 您会惊喜地发现，您心仪的最新文字，就已经如约在全球范围内同步生效了！✨
